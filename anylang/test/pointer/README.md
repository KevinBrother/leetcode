# 指针

## 值类型和引用类型区别

引用类型的变量存储的是指向实际数据的地址，而不是数据本身，修改一个变量的值会影响另一个变量的值。
值类型是值值的拷贝，修改一个变量值不会影响另一个变量

### go

- 值类型
    array、string、基本类型
- 引用类型
    slice(切片)、map、channel、pointer 是，

- tip
    go 中 array 是值类型，且数组的长度是类型的一部分

### js

- 值类型
    基本类型
- 引用类型
    复杂类型

### C/C++

- 值类型
    基本类型
- 引用类型
    array、等复杂类型
